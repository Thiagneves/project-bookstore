# 📚 OpenBook

O **OpenBook** é um projeto prático de desenvolvimento web criado com o objetivo de demonstrar minhas habilidades em front-end e, futuramente, back-end.

A proposta do sistema é simular uma livraria online onde usuários podem navegar por livros, adicionar itens ao carrinho e realizar o aluguel de forma simples.

---

## 🚀 Funcionalidades atuais

- 📖 Navegação por páginas de livros
- 🛒 Adição de livros ao carrinho
- 📦 Simulação de aluguel de livros
- 📞 Página de contato com a empresa
- ⚡ Interações com JavaScript

---

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript
